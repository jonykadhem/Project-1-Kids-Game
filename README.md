# Project-1-Kids-Game
A simple education game for kids to help them learn about:
- alphabets
- numbers
- colors
- animals

![screen shot of the game](assets\Capture.PNG)

## Getting Started

### play The Game
visit [the game](https://jonykadhem.github.io/Project-1-Kids-Game/)

### How To Play
1. Choose a category.
2. Click "Start"
3. Choose the answers
4. Compleat all questions to win
### Installation 
```bash
git clone https://github.com/jonykadhem/Project-1-Kids-Game.git
cd Project-1-Kids-Game
open main.html
```
